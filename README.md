
# GuardAI: Neural Wellness System

GuardAI is a wellness system that uses an empathetic avatar to visualize student fatigue in real-time, preventing burnout with smart wellness interventions before it affects learning.

##  The Problem
Students lose up to 40% of learning efficiency in the final hour of study because they don't notice their own fatigue. Traditional tools only track time—they don't track focus or energy levels.

##  The Solution
GuardAI uses an empathetic avatar that changes with your fatigue level. When fatigue builds up, the system intervenes with wellness breaks before burnout occurs.

##  Features
-  **Empathetic Avatar** - Changes expression based on fatigue level
-  **Real-time Energy Bar** - Visual indicator of remaining focus energy
-  **Smart Alert System** - Triggers at critical fatigue levels (75%)
-  **Wellness Interventions** - Power Nap, Box Breathing, Movement breaks
-  **Lockdown Protection** - Forces break after 3 ignored alerts
-  **Wellness Dashboard** - Tracks fatigue patterns and burnout risk

##  Built With
- HTML5, CSS3, JavaScript (ES6)
- Chart.js for data visualization
- LocalStorage for data persistence

##  How to Try It
1. Clone this repository
2. Open `index.html` in any modern browser
3. Click "Start Session"
4. Click "Simulate Fatigue" to see the avatar react
5. Watch the alert trigger at 75% fatigue
